# Game on Obsidian

一个关于"在 Obsidian 的 Markdown 文档里玩文字游戏"的概念探索项目。

灵感源：
- **Loot Project**（2021 加密圈纯文本装备 NFT）
- **Rare Pepe**（早期 Counterparty 文字/图像收藏卡）
- **Claude Buddy**（Anthropic 2026 年 4 月推出的终端电子鸡）

核心信念：**Markdown 文件即一级游戏存档**。状态在外部、文本即接口、留白激发创作。

## 当前阶段

概念探索期。已完成：
- 三大灵感源的关键洞察调研
- 三个候选 MVP 方向的设计与对比
- **每个方向的纯静态展示厅**（见 `samples/`）
