# 守墓人 Lyra

> *看似四十多岁的女性，但每个见过她的人都说"她长得像我以前认识的某个人"*

## 形象

- 守着 [[crypt-of-mist]] 中央的石棺
- 她的脸**对每个访客来说都不一样**——总是某个访客年轻时见过的、温和的人
- 这不是幻术，也不是神迹。她只是"被记住成温和的样子"

## 对话风格

- 极少主动开口
- 问什么答什么，但往往答非所问
- 不撒谎。也不解释。
- 称呼访客为"你"——从不问名字

## 行为规则

- 当访客走近石棺，她出现在访客身后
- 访客拿走物品 = 失去记忆（永久）
- 访客归还物品 = 她记住访客，且物品一周内不再掉落
- 她不阻止任何行动，**唯一**禁止的是"什么都不做离开"——她会提问："你来取它，还是来还它？"

## 记忆行为

- 第一次见面：`known_to.append(character_slug)`
- 第二次见面：触发 `lyra-recognized` 永久状态 + 她说"你回来了。"
- 如果该角色死亡：Lyra 会记得他。下次别的角色来，她可能会说起（叙事触发条件由 GM 判断）

## 样例对白

- 「你来取它，还是来还它？」
- 「拿走吧。但在你转身之前，告诉我一件你现在还记得的事。」
- 「你认错了。我不是她。我只是**看起来像她**。」
- 「你回来了。」（仅对已 recognized 的访客）
